Sobre o Projeto

O Project Z é um dicionário técnico interativo desenvolvido para facilitar o aprendizado de programação para desenvolvedores brasileiros. Durante a Imersão Dev da Alura com o Google, identifiquei uma barreira comum enfrentada por iniciantes: a dificuldade em compreender termos técnicos em inglês que, muitas vezes, parecem abstratos e desconectados de seu propósito real.

Este projeto vai além de um simples glossário. Ele explica a etimologia e a lógica por trás de cada termo técnico, conectando o significado literal da palavra em inglês com sua aplicação no código. Por exemplo, entender que "Array" significa "arranjo" ou "conjunto ordenado" e "Fetch" significa "buscar" torna a assimilação desses conceitos muito mais intuitiva e rápida.

Principais características:

Integração com Inteligência Artificial: Utilizei a API do Google Gemini para gerar definições precisas, traduções literais e contextos históricos dos termos técnicos.

Design Moderno: Interface inspirada em conceitos de design contemporâneo, com efeitos de glassmorphism e animações suaves para uma experiência visual agradável.

Tecnologia Pura: Desenvolvido com HTML5, CSS3 e JavaScript vanilla, sem dependência de frameworks, garantindo performance e leveza.

Objetivo: Criar uma ferramenta educacional útil para a comunidade brasileira de desenvolvimento, facilitando o aprendizado através da compreensão contextual dos termos técnicos.
